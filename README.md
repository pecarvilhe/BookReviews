## 📊 Dashboard de Avaliações de Livros

Projeto simples em Python para visualização de dados de livros e reviews, utilizando as bibliotecas:

- Pandas  
- Streamlit  
- Plotly  

---

## 🚀 O que faz

- Filtra livros por preço  
- Mostra gráficos de:
  - Publicações por ano  
  - Distribuição de preços  
- Exibe detalhes e avaliações de um livro selecionado  

---

## ▶️ Como executar

```bash
pip install streamlit pandas plotly
streamlit run main.py
```
